### フッターのカスタマイズ

　現在のフッターは、表示物を減らしたら、だいぶ間抜けている感じだ。

![](/2025-12-12/J2lAv9wgugRFShpxMF3huVGw.png)

　トップと同様でアイコンとメニューを横並びでいいんじゃないかな？

　フォーマットはfooter.htmlが決めている。  
　footer.htmlでは、「ソーシャルアイコンのブロック」と「メニューのブロック」が別々のブロックに書かれている。これらを display: flex というスタイルを適用した箱で囲んでしまう。  
　アイコン群とメニュー群の間の距離はgapで調整。  
　flex-wrap: wrap;でスマホなど画面が狭いときに、無理やり横並びせずに自動で縦積みに戻して画面が崩れるのを予防する。  
    style="margin-bottom: 0;"で、元のテーマのCSSでulタグに下の余白がついているので、横並びにしたときに高さがズレないよう余白を打ち消す。

```
% cp themes/vonge/templates/partials/footer.html templates/partials/
% cat templates/partials/footer.html
<!-- begin footer -->
<footer class="footer">
  <div class="container">
    <div class="row">
      <div class="col col-12">

        {% include "partials/social-link.html" %}

        <ul class="footer_nav list-reset">
          {% for item in config.extra.footer.menu %}
          {% set url = item.url | replace(from='$BASE_URL', to=config.base_url) %}
            <li class="footer_nav__item">
              <a href="{{ url }}"
                class="footer_nav__link">{{ item.title }}</a>
            </li>
          {% endfor %}
        </ul>

        {% if config.extra.footer.copyright %}
        <div class="copyright">{{ config.extra.footer.copyright }}</div>
        {% else %}
        <div class="copyright">&copy; {{ now() | date(format="%Y") }} <a href="{{ config.base_url }}">{{ config.author }}</a></div>
        {% endif %}

      </div>
    </div>
  </div>
</footer>
<!-- end footer -->
```

　修正。

```
<!-- begin footer -->
<footer class="footer">
  <div class="container">
    <div class="row">
      <div class="col col-12">

        <!-- ▼▼▼ 横並び用の箱で囲む開始 ▼▼▼ -->
        <div class="footer-flex-container" style="display: flex; justify-content: center; align-items: center; gap: 40px; flex-wrap: wrap; margin-bottom: 32px;">

            <!-- 1. ソーシャルアイコン -->
            <div class="footer-social-wrapper">
                {% include "partials/social-link.html" %}
            </div>

            <!-- 2. フッターメニュー -->
            <!-- margin-bottom: 0 を指定して、高さのズレを防ぎます -->
            <ul class="footer_nav list-reset" style="margin-bottom: 0;">
              {% for item in config.extra.footer.menu %}
              {% set url = item.url | replace(from='$BASE_URL', to=config.base_url) %}
                <!-- 必要であれば li に margin-left などを足してメニュー間の距離を調整できます -->
                <li class="footer_nav__item">
                  <a href="{{ url }}" class="footer_nav__link">{{ item.title }}</a>
                </li>
              {% endfor %}
            </ul>

        </div>
        <!-- ▲▲▲ 横並び用の箱で囲む終了 ▲▲▲ -->

        {% if config.extra.footer.copyright %}
        <div class="copyright">{{ config.extra.footer.copyright }}</div>
        {% else %}
        <div class="copyright">&copy; {{ now() | date(format="%Y") }} <a href="{{ config.base_url }}">{{ config.author }}</a></div>
        {% endif %}

      </div>
    </div>
  </div>
</footer>
<!-- end footer -->
```

![](/2025-12-12/GwwfeBh4XhURS8YW3KAo8KP1.png)

　アイコンとメニューを高さの中央で揃えたい。全体の高さを低くしたい。  
    footer タグに padding: 20px 0; を指定して、上下の余白を縮める。  
　style タグを追加して、フッター内のすべてのリスト（ul）やブロックの余白を強制的にゼロにする。  
　フレックスボックスにalign-items: centerしておく。

　だいぶスリムになった。

![](/2025-12-12/bdN4oWKyKEzseD8Dl6kNLGwQ.png)

### 背景グラデーションの変更

　背景のグラデーションの色がいかにもだな。茶色系にするかな。

　VONGEでは、2つのメインカラーであるPrimaryとSecondaryを設定するだけで、サイト全体のグラデーション（ヒーローヘッダー、ボタン、タグなど）が自動的に生成される仕組みになっている。  
　設定は、sass/css/0-settings/_B_color-scheme.scssにある。

```
% vi sass/css/0-settings/_B_color-scheme.scss
```

　デフォルトはこれ。

```
/* Color Scheme */
$primary-color: #1495a7;
$secondary-color: #003687;
```

　濃いめの茶色。ビター・チョコレートという組み合わせだけど、ミルクチョコに近いか。

```
$primary-color: #4e342e;   /* 濃い焦げ茶 */
$secondary-color: #8d6e63; /* ミルクチョコ色 */
```

![](/2025-12-12/PTK5tPFqwjb5c0xhrANnTG86.png)

　カフェ・ラテ。ちょっと白いかな。

```
$primary-color: #8d6e63;   /* 落ち着いた茶色 */
$secondary-color: #d7ccc8; /* ベージュに近い薄茶 */
```

![](/2025-12-12/9YW2PvXWENgw2jmtljq9AOw6.png)

　レザー・アンバー。

```
$primary-color: #a1887f;   /* グレイッシュな茶色 */
$secondary-color: #ffb74d; /* 明るいアンバー */
```

![](/2025-12-12/jZmISEjvKpX9vnpyrlPtG5M2.png)

　白文字の視認性が高いので、ビター・チョコレートにする。

### 画像のセンタリング

　ようやく本文にたどりついた。

　Blogの本文に画像を入れると左寄せになってしまう。これを中央にしたい。

![](/2025-12-12/h6IWvNPzuL5JTl85CPBMY08Q.png)

　scssを変更する。  
　.post-image {}かな？

```
% vi sass/css/4-layouts/_page-layout.scss
```

　末尾に追記する。

```
.post__content img {
  display: block;           /* ブロック要素にする */
  margin-left: auto;        /* 左余白を自動 */
  margin-right: auto;       /* 右余白を自動 */
  margin-bottom: 30px;      /* 画像の下に少し余白を入れると読みやすいです */
  max-width: 100%;          /* スマホではみ出さないようにする */
  height: auto;             /* 比率を維持する */
}
```

![](/2025-12-12/XuTUMzcRvH4FIq1aGuhjRUBl.png)

　できた。
