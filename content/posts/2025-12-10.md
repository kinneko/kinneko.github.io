　でも、これをデカくしてしまうと、Blogの画面のタグもデカくなってしまうのな。それはよろしくない。.c-tags-pageを新しくコピペで作って、タグページのほうもそれを使うように変更する。

```
% vi sass/css/components/_blog-card.scss
```

　末尾に以下を追記。

```
/* ------------------------------------------------ */
/* タグ一覧ページ専用のスタイル (c-tags-page) */
/* ------------------------------------------------ */
.c-tags-page {
  &__tags-box {
    /* 必要ならここにボックス全体の設定 */
  }

  &__tag {
    position: relative;
    z-index: 1;
    display: inline-block;
    
    /* ▼▼▼ ここを大きくしました ▼▼▼ */
    padding: 10px 20px;     /* 箱を大きく */
    margin: 0 8px 8px 0;    /* 間隔を広く */
    font-size: 18px;        /* 文字サイズ大 */
    line-height: 1.5;       /* 行間 */
    /* ▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲▲ */

    font-weight: 600;
    text-transform: capitalize;
    border-radius: 8px;
    overflow: hidden;
    color: $white; /* 文字色 */
    
    /* 背景色（元の色と同じグラデーション） */
    background: linear-gradient(90deg, $primary-color 0%, $secondary-color 167%);

    &:hover {
      &:before {
        opacity: 1;
      }
    }

    &::before {
      content: "";
      position: absolute;
      top: 0;
      left: 0;
      z-index: -1;
      display: block;
      width: 100%;
      height: 100%;
      opacity: 0;
      border-radius: 8px;
      will-change: transform;
      transition: opacity 0.35s;
      background: linear-gradient(90deg, $primary-color 0%, $secondary-color 107%);
    }

    &:last-child {
      margin: 0 0 8px 0;
    }
  }
}
```

　{% for ... %} のループの中で、アイコン名が「Twitter」のときだけ SVG を表示し、それ以外は元の <i class="ion ..."> を表示するように分岐させる。

　これを。

```
    {% for social_icon in config.extra.social_icons %}
    <li class="social__item">
      <a class="social__link" href="{{ social_icon.url | replace(from='$BASE_URL', to=config.base_url) }}" target="_blank" rel="noopener"
        aria-label="{{ social_icon.icon }} icon"><i class="ion ion-logo-{{ social_icon.icon | lower }}"></i></a>
```

