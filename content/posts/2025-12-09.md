　vongeをさわっていると、なんとなく設計思想がわかってくる感じ。

　基本ルールとしては、themesフォルダの中はまったく編集せずに、中身を継承して使うということか。　  
　手順にはなかったけど、まずはthemes/vonge/config.tomlをZolaのトップにコピーする。中身がほとんど同じのtheme.tomlというのもあるけど、関連度はわからない。

```
% cp themes/vonge/config.toml ./
```

　インストール時に、themes/vonge/contentの中身をトップのcontentにコピーしているので、自前のコンテンツはこの中に置いて使う。  
　こんな感じかな。  
　config.tomlの中に[[extra.content_blocks]]があるので、それで表示している内容を並べている。不要なブロックはここから削除するだけで消える。

　Blogの個別ページとタグの_index.htmlはどこにあるのだろう？

　Blogのほうは、themes/vonge/templates/page.htmlのようだ。テーマは直接はさわらないのが鉄則なので、これをtemplates/page.htmlにコピーして編集する。２行を削除したらよさそう。

```
% vi templates/page.html
{% import "components/newsletter.html" as newsletter %}

{{ newsletter::newsletter(config = config.extra.newsletter) }}
```

　無事消えたけど、「You may also like」もいらないかな...

![](/2025-12-09/ZtPrSTefDss8p4Sg2gnHoNgo.png)

　themes/vonge/templates/partials/related-posts.htmlに文字列があるので、こいつかな？

```
{% include "partials/related-posts.html" %}
```

　削除したら消えた。

![](/2025-12-09/Nn8Xcvy0XChTJ89UFc79smar.png)
